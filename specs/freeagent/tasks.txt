<h2 id="list-all-tasks">List all tasks</h2><pre class=prettyprint><code class=>GET https://api.freeagent.com/v2/tasks
</code></pre><h3>Input</h3><h4>View Filters</h4><pre class=prettyprint><code class=>GET https://api.freeagent.com/v2/tasks?view=active
</code></pre>
<ul>
<li><code>all</code>: Show all tasks (default)</li>
<li><code>active</code>: Show only tasks with status active.</li>
<li><code>completed</code>: Show only tasks with status completed.</li>
<li><code>hidden</code>: Show only tasks with status hidden.</li>
</ul>
<h4>Date Filters</h4><pre class=prettyprint><code class=>GET https://api.freeagent.com/v2/tasks?updated_since=2017-04-06
</code></pre>
<ul>
<li><code>updated_since</code></li>
</ul>
<h4>Sort Orders</h4><pre class=prettyprint><code class=>GET https://api.freeagent.com/v2/tasks?sort=updated_at
</code></pre>
<ul>
<li><code>name</code>: Sort by the task name (default).</li>
<li><code>project</code>: Sort by the project_id associated with the task.</li>
<li><code>billing_rate</code>: Sort by the billing rate.</li>
<li><code>created_at</code>: Sort by the time the task was created.</li>
<li><code>updated_at</code>: Sort by the time the task was last modified.</li>
</ul>
