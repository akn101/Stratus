          <h1>Stock Items</h1>
<p><em>Minimum access level</em>: <code>Invoices, Estimates and Files</code></p>

<p>Provides read-only access to the list of stock items defined in the FreeAgent account.</p>
<h2 id="attributes">Attributes</h2>
<table>
  <thead>
    <tr>
      <th>Attribute</th>
      <th>Description</th>
      <th>Kind</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>url</td>
      <td>The unique identifier for the stock item</td>
      <td>URI</td>
    </tr>
    <tr>
      <td>description</td>
      <td>
        Free-text description / code to identify the item when it's added to an <a href="invoices.html">invoice</a> or <a href="estimates.html">estimate</a>
      </td>
      <td>String</td>
    </tr>
    <tr>
      <td>opening_quantity</td>
      <td>Stock on hand as of FreeAgent Start Date</td>
      <td>Integer</td>
    </tr>
    <tr>
      <td>opening_balance</td>
      <td>The value of stock on hand as of FreeAgent Start Date</td>
      <td>Decimal</td>
    </tr>
    <tr>
      <td>cost_of_sale_category</td>
      <td>The <a href="categories.html">spending category</a> which will be used to account for sales of this stock item</td>
      <td>URI</td>
    </tr>
    <tr>
      <td>stock_on_hand</td>
      <td>Stock on hand as of today</td>
      <td>Integer</td>

      <td>Timestamp</td>

<h2 id="list-all-stock-items">List all Stock Items</h2><pre class=prettyprint><code class=>GET https://api.freeagent.com/v2/stock_items
</code></pre><h4>Sort Orders</h4><pre class=prettyprint><code class=>GET https://api.freeagent.com/v2/stock_items?sort=created_at
</code></pre>
<ul>
<li><code>created_at</code>: Sort by the time the stock item was created (default).</li>
<li><code>description</code>: Sort by the stock item description / code.</li>
<li><code>updated_at</code>: Sort by the time the stock item was updated.</li>
</ul>

<p>To sort in descending order, the sort parameter can be prefixed with a hyphen.</p>
<pre class=prettyprint><code class=>GET https://api.freeagent.com/v2/stock_items?sort=-created_at
