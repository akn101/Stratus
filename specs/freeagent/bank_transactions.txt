          <h1>Bank Transactions</h1>
<p><em>Minimum access level</em>: <code>Banking</code>, unless stated otherwise.</p>

      <td>The unique identifier for the bank transaction</td>
      <td>URI</td>

    <tr>

      <td>URI</td>
    </tr>

      <td>Date of transaction in <code>YYYY-MM-DD</code> format</td>
      <td>Date</td>
    </tr>
    <tr>
      <td>description</td>
      <td>Free-text description</td>
      <td>String</td>
    </tr>
    <tr>
      <td>full_description</td>
      <td>A complete description of the transaction, including the amount and type</td>
      <td>String</td>
    </tr>
    <tr>
      <td>uploaded_at</td>
      <td>When the transaction was uploaded to FreeAgent</td>
      <td>Timestamp</td>
    </tr>
    <tr>
      <td>unexplained_amount</td>
      <td>Amount yet to be explained</td>
      <td>Decimal</td>
    </tr>

    </tr>

      <td>transaction_id</td>

      <td>String</td>

    <tr>

      <td>Creation of the bank transaction (UTC)</td>
      <td>Timestamp</td>
    </tr>
    <tr>
      <td>updated_at</td>
      <td>When the bank transaction was last updated (UTC)</td>
      <td>Timestamp</td>
    </tr>

<h2 id="list-all-bank-transactions-under-a-certain-bank-account">List all bank transactions under a certain bank account</h2><pre class=prettyprint><code class=>GET https://api.freeagent.com/v2/bank_transactions?bank_account=:bank_account
</code></pre><h4>Date Filters</h4><pre class=prettyprint><code class=>GET https://api.freeagent.com/v2/bank_transactions?bank_account=:bank_account&amp;from_date=2012-01-01&amp;to_date=2012-03-31
</code></pre><pre class=prettyprint><code class=>GET https://api.freeagent.com/v2/bank_transactions?bank_account=:bank_account&amp;updated_since=2017-05-22T09:00:00.000Z
</code></pre>
<ul>
<li><code>from_date</code></li>
<li><code>to_date</code></li>
<li><code>updated_since</code></li>
</ul>
<h4>View Filters</h4><pre class=prettyprint><code class=>GET https://api.freeagent.com/v2/bank_transactions?bank_account=:bank_account&amp;view=unexplained
</code></pre>
<ul>
<li><code>all</code> (default)</li>
<li><code>unexplained</code></li>
<li><code>explained</code></li>
<li><code>manual</code></li>
<li><code>imported</code></li>
<li><code>marked_for_review</code></li>
</ul>
<h4>Filtering by latest statement upload</h4>
<p>Returns only transactions that are part of the last statement that has been uploaded.
Note that this may have been uploaded via the API or by a user via the web interface.</p>
<pre class=prettyprint><code class=>GET https://api.freeagent.com/v2/bank_transactions?bank_account=:bank_account&amp;last_uploaded=true
