          <h1>Categories</h1>
<p><em>Minimum access level</em>: <code>My Money</code>, unless stated otherwise.</p>

<p>Categories are used to explain the type of income or cost for estimate items, invoice items, expenses, bills and bank transaction explanations.</p>

<h2 id="list-all-categories">List all categories</h2>
<p>Returns the list of all categories for the current company in four sets:
Admin Expenses, Cost of Sales, Income and General.  The list of
categories varies between companies as users can create custom
categories.</p>
<pre class=prettyprint><code class=>GET https://api.freeagent.com/v2/categories
</code></pre><h3>Input</h3><h4>List all categories including sub accounts</h4>
<p>Returns the list of all categories as above except it includes sub accounts
in the list instead of the associated top level accounts where they exist.
Sub accounts can be identified by inspecting the nominal code which will
include a - sign and a sub code - for example <code>602-1</code>.</p>
<pre class=prettyprint><code class=>GET https://api.freeagent.com/v2/categories?sub_accounts=true
</code></pre><h3>Response</h3><pre class=prettyprint><code class=>Status: 200 OK
</code></pre>
