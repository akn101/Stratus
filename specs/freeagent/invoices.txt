<h2 id="list-all-invoices">List all invoices</h2><pre class=prettyprint><code class=>GET https://api.freeagent.com/v2/invoices
</code></pre><h3>Input</h3><h4>View Filters</h4><pre class=prettyprint><code class=>GET https://api.freeagent.com/v2/invoices?view=recent_open_or_overdue
</code></pre>
<ul>
<li><code>all</code>: Show all invoices (default)</li>
<li><code>recent_open_or_overdue</code>: Show only recent, open, or overdue invoices.</li>
<li><code>open</code>: Show only open invoices.</li>
<li><code>overdue</code>: Show only overdue invoices.</li>
<li><code>open_or_overdue</code>: Show only open or overdue invoices.</li>
<li><code>draft</code>: Show only draft invoices.</li>
<li><code>paid</code>: Show only paid invoices.</li>
<li><code>scheduled_to_email</code>: Show only invoices scheduled to email.</li>
<li><code>thank_you_emails</code>: Show only invoices with active thank you emails.</li>
<li><code>reminder_emails</code>: Show only invoices with active reminders.</li>
<li><code>last_N_months</code>: Show only invoices from the last <code>N</code> months.</li>
</ul>
<h4>Date Filters</h4><pre class=prettyprint><code class=>GET https://api.freeagent.com/v2/invoices?updated_since=2017-05-22T09:00:00.000Z
</code></pre>
<ul>
<li><code>updated_since</code></li>
</ul>
<h4>Sort Orders</h4><pre class=prettyprint><code class=>GET https://api.freeagent.com/v2/invoices?sort=updated_at
</code></pre>
<ul>
<li><code>created_at</code>: Sort by the time the invoice was created (default).</li>
<li><code>updated_at</code>: Sort by the time the invoice was last modified.</li>
</ul>

<p>To sort in descending order, the sort parameter can be prefixed with a hyphen.</p>
