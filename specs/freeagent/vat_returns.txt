          <h1>VAT Returns</h1>
<p><em>Minimum access level</em>: <code>Tax, Accounting &amp; Users</code>, unless stated otherwise.</p>

<p>This endpoint allows API applications to retrieve information on a company&#39;s VAT Returns, including details on payments due and filing status.</p>
<h2 id="attributes">Attributes</h2>
<table><thead>
<tr>
<th>Attribute</th>
<th>Description</th>
<th>Kind</th>
</tr>
</thead><tbody>
<tr>
<td>url</td>
<td>Unique identifier for the record</td>
<td>URI</td>
</tr>
<tr>
<td>period_ends_on</td>
<td>The end date of the period covered by the return</td>
<td>Date</td>
</tr>
<tr>
<td>period_starts_on</td>
<td>The start date of the period covered by the return</td>
<td>Date</td>
</tr>
<tr>
<td>payments</td>
<td>An array of <a href="#payment-attributes">Payments</a></td>
<td>Array</td>
</tr>
<tr>
<td>filing_due_on</td>
<td>The date on which submission is due</td>
<td>Date</td>
</tr>
<tr>
<td>filing_status</td>
<td>One of <code>unfiled</code>, <code>pending</code>, <code>rejected</code>, <code>filed</code> or <code>marked_as_filed</code>.</td>
<td>String</td>
</tr>
<tr>
<td>filed_at</td>
<td>For returns which have been filed online, the date and time at which that happened</td>
<td>Datetime</td>
</tr>
<tr>
<td>filed_reference</td>
<td>For returns which have been filed online, the form bundle number returned by the submission</td>
<td>String</td>
</tr>
</tbody></table>
<h2 id="payment-attributes">Payment Attributes</h2>
<table><thead>
<tr>
<th>Attribute</th>
<th>Description</th>
<th>Kind</th>
</tr>
</thead><tbody>
<tr>
<td>label</td>
<td>A human-readable name for the payment</td>
<td>String</td>
</tr>
<tr>
<td>due_on</td>
<td>The date on which a payment is due</td>
<td>Date</td>
</tr>
<tr>
<td>amount_due</td>
<td>The amount due on that date. May be negative in the case of a refund.</td>
<td>Decimal</td>
</tr>
<tr>
<td>status</td>
<td>One of <code>unpaid</code> or <code>marked_as_paid</code>. <code>paid</code> is reserved for future use. Key is omitted if <code>amount_due</code> is zero or negative.</td>
<td>String</td>
</tr>
</tbody></table>
