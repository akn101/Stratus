<h2 id="list-all-contacts">List all contacts</h2><pre class=prettyprint><code class=>GET https://api.freeagent.com/v2/contacts
</code></pre><h3>Input</h3><h4>View Filters</h4><pre class=prettyprint><code class=>GET https://api.freeagent.com/v2/contacts?view=active
</code></pre>
<ul>
<li><code>all</code>: Show all contacts.</li>
<li><code>active</code>: Show only active contacts (default).</li>
<li><code>clients</code>: Show all clients.</li>
<li><code>suppliers</code>: Show only active suppliers.</li>
<li><code>active_projects</code>: Show only clients with active projects.</li>
<li><code>completed_projects</code>: Show only clients with completed invoices.</li>
<li><code>open_clients</code>: Show only clients with open invoices.</li>
<li><code>open_suppliers</code>: Show only suppliers with open bills.</li>
<li><code>hidden</code>: Show only hidden contacts.</li>
</ul>
<h4>Sort Orders</h4><pre class=prettyprint><code class=>GET https://api.freeagent.com/v2/contacts?sort=updated_at
</code></pre>
<ul>
<li><code>name</code>: Sort by the concatenation of <code>organisation_name</code>, <code>last_name</code> and <code>first_name</code> (default).</li>
<li><code>created_at</code>: Sort by the time the contact was created.</li>
<li><code>updated_at</code>: Sort by the time the contact was last modified.</li>
</ul>

<p>To sort in descending order, the sort parameter can be prefixed with a hyphen.</p>
<pre class=prettyprint><code class=>GET https://api.freeagent.com/v2/contacts?sort=-updated_at
</code></pre><h4>Date Filters</h4><pre class=prettyprint><code class=>GET https://api.freeagent.com/v2/contacts?updated_since=2025-03-15T09:00:00.000Z
</code></pre>
<ul>
<li><code>updated_since</code>: Only return contacts that have been updated since the provided date and time (format ISO 8601).</li>
</ul>
