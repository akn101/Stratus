<h2 id="list-all-projects">List all projects</h2><pre class=prettyprint><code class=>GET https://api.freeagent.com/v2/projects
</code></pre><h3>Input</h3><h4>View Filters</h4><pre class=prettyprint><code class=>GET https://api.freeagent.com/v2/projects?view=active
</code></pre>
<ul>
<li><code>active</code>: Show only active projects.</li>
<li><code>completed</code>: Show only completed projects.</li>
<li><code>cancelled</code>: Show only cancelled projects.</li>
<li><code>hidden</code>: Show only hidden projects.</li>
</ul>
<h4>Sort Orders</h4><pre class=prettyprint><code class=>GET https://api.freeagent.com/v2/projects?sort=updated_at
</code></pre>
<ul>
<li><code>name</code>: Sort by project name (default).</li>
<li><code>contact_name</code>: Sort by the concatenation of <code>organisation_name</code>, <code>last_name</code> and <code>first_name</code> of the contact.</li>
<li><code>contact_display_name</code>: Sort by the concatenation of <code>organisation_name</code>, <code>first_name</code> and <code>last_name</code> of the contact.</li>
<li><code>created_at</code>: Sort by the time the project was created.</li>
<li><code>updated_at</code>: Sort by the time the project was last modified.</li>
</ul>

<p>To sort in descending order, the sort parameter can be prefixed with a hyphen.</p>
<pre class=prettyprint><code class=>GET https://api.freeagent.com/v2/projects?sort=-updated_at
</code></pre><h4>Nested Response</h4><pre class=prettyprint><code class=>GET https://api.freeagent.com/v2/projects?nested=true
</code></pre>
<ul>
<li><code>true</code>: Return full contact details for each project as a nested json object.  See <a href="contacts.html">Contact</a></li>
<li><code>false</code>: Return a RESTful reference to the contact for each project and the contact&#39;s display name</li>
</ul>
