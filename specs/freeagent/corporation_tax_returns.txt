          <h1>Corporation Tax Returns</h1>
<p><em>Minimum access level</em>: <code>Tax, Accounting &amp; Users</code>, unless stated otherwise.</p>

<p>This endpoint allows API applications to retrieve information on a company&#39;s Corporation Tax Returns, including amount due, deadlines and filing/payment status.</p>

<tr>
<th>Attribute</th>
<th>Description</th>
<th>Kind</th>

</tr>
<tr>

<td>The end date of the period covered by the return</td>
<td>Date</td>
</tr>
<tr>
<td>period_starts_on</td>
<td>The start date of the period covered by the return</td>
<td>Date</td>
</tr>
<tr>
<td>amount_due</td>
<td>Amount to be paid by <code>payment_due_on</code></td>
<td>Decimal</td>
</tr>
<tr>
<td>payment_due_on</td>
<td>The date on which payment is due</td>
<td>Date</td>
</tr>
<tr>
<td>payment_status</td>
<td>For returns which require payment, one of <code>unpaid</code> or <code>marked_as_paid</code>. <code>paid</code> is reserved for future use. Key is omitted if there are no payments to be made.</td>
<td>String</td>
</tr>
<tr>
<td>filing_due_on</td>
<td>The date on which submission is due</td>

</tr>
