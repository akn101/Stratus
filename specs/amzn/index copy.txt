<h2 id="list-all-estimates">List all estimates</h2><pre class=prettyprint><code class=>GET https://api.freeagent.com/v2/estimates
</code></pre><h3>Input</h3><h4>View Filters</h4><pre class=prettyprint><code class=>GET https://api.freeagent.com/v2/estimates?view=recent
</code></pre>
<ul>
<li><code>all</code>: (default)</li>
<li><code>recent</code>: Show only recent estimates.</li>
<li><code>draft</code>: Show only estimates marked as draft.</li>
<li><code>non_draft</code>: Show only estimates the not marked as draft.</li>
<li><code>sent</code>: Show only estimates marked as sent.</li>
<li><code>approved</code>: Show only estimates marked as approved.</li>
<li><code>rejected</code>: Show only estimates marked as rejected.</li>
<li><code>invoiced</code>: Show only estimates marked as invoiced.</li>
</ul>
<h4>Date Filters</h4><pre class=prettyprint><code class=>GET https://api.freeagent.com/v2/estimates?from_date=2012-01-01&amp;to_date=2012-03-21
</code></pre><pre class=prettyprint><code class=>GET https://api.freeagent.com/v2/estimates?updated_since=2017-05-22T09:00:00.000Z
</code></pre>
<ul>
<li><code>from_date</code></li>
<li><code>to_date</code></li>
<li><code>updated_since</code></li>
</ul>
